Ballspiel

Das Ziel dieses kleinen Spiels ist es, möglichst viele Punkte zu erreichen (Wie so oft ;-). Dies ist am besten dadurch zu bewerkstelligen, möglichst viele Bälle, durch einen Mausclick auf dieselbigen, abzuschießen. Wenn sie einen Ball ins Aus gehen lassen, ohne ihn getroffen zu haben, so verlieren sie eines von zehn Leben. Sollten keine Leben mehr übrig sein, so ist das Spiel vorbei und ihnnen wird mitgeteilt, wieviele Prozent ihrer Schüsse getroffen haben. Außerdem wird die von ihnen erreichte Punktzahl bewertet (Höchstwertung: "You are the Champingnon!"). Viel Spaß beim Spielen und viel Erfolg!
